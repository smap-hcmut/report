#### 4.7.7.4 Cost Analysis (Technology TCO)

**Total Cost of Ownership - So sánh với phương án khác:**

| Component | SMAP Choice (On-premise) | Cloud Alternative (AWS) | Savings |
|-----------|-------------------------|------------------------|---------|
| **Compute** | 3 nodes × $200/month = $600 | EKS (48 pods) = $1,800/month | **-67%** |
| **Databases** | PostgreSQL + MongoDB (self-hosted) = $0 | RDS + DocumentDB = $500/month | **-100%** |
| **Storage** | MinIO (500GB) = $50/month | S3 (500GB + egress) = $200/month | **-75%** |
| **Message Broker** | RabbitMQ (self-hosted) = $0 | Amazon MQ = $150/month | **-100%** |
| **Monitoring** | Prometheus + Grafana (free) = $0 | Datadog = $300/month | **-100%** |
| **Total** | **$650/month** | **$2,950/month** | **-78%** |

**Lưu ý:**
- Chi phí trên chưa tính nhân công vận hành
- On-premise tiết kiệm chi phí nhưng cần team DevOps
- Cloud dễ scale nhưng có vendor lock-in

#### 4.7.7.8 Summary & Key Decisions

**Tóm tắt các quyết định công nghệ quan trọng:**

✅ **Polyglot Architecture (Golang + Python):**
- Go cho API services (fast, low memory)
- Python cho AI/ML (rich ecosystem)
- **Evidence:** API Gateway 15k rps, Analytics 650 posts/sec

✅ **Polyglot Persistence (PostgreSQL + MongoDB + Redis):**
- PostgreSQL cho structured data (ACID)
- MongoDB cho raw data (schema-less, write-heavy)
- Redis cho cache (sub-ms latency)
- **Evidence:** Query time <50ms, Cache hit 82%

✅ **Self-hosted Infrastructure (K8s + RabbitMQ + MinIO):**
- Cost savings: -78% vs Cloud
- No vendor lock-in
- **Evidence:** $650/month vs $2,950/month (AWS)

✅ **Open-source First:**
- No licensing costs
- Huge community support
- Easy to customize
- **Evidence:** Total software license cost: $0
